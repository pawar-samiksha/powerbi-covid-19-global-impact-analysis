# COVID-19 Global Impact Analysis Dashboard

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze the global impact of the COVID-19 pandemic through data visualization and trend analysis.

The dashboard explores worldwide COVID-19 cases, recoveries, deaths, active cases, and case fatality trends across different countries and time periods. The project combines storytelling with analytics to provide a comprehensive understanding of how the pandemic evolved globally.

---

## Project Objectives
- Analyze global COVID-19 case distribution
- Compare recoveries and deaths across countries
- Track monthly pandemic trends and fatality rates
- Visualize the geographical spread of COVID-19
- Identify countries and periods with high pandemic impact
- Transform raw pandemic data into meaningful visual insights

---

## Dashboard Features

### 1️ Global KPI Overview
The dashboard provides high-level pandemic statistics including:
- Total Cases
- Total Recoveries
- Total Deaths
- Recovery Rate
- Case Fatality Rate

#### Key Insights
- Billions of COVID-19 cases and recoveries were reported worldwide
- The pandemic affected nearly every region globally
- Recovery rates improved over time despite rising case counts

---

### 2️ Country-Level Case Analysis
Bar charts compare the most affected countries based on:
- Total Cases
- Recoveries
- Deaths

#### Key Insights
- The United States reported the highest number of cases
- India recorded the highest recoveries
- Brazil and Mexico showed significant mortality figures

---

### 3️ Monthly Trend Analysis
Line and area charts analyze:
- Monthly deaths
- Monthly case fatality rates
- Pandemic wave progression

#### Key Insights
- Early 2021 showed a peak in Case Fatality Rate
- Monthly death patterns reflected multiple pandemic waves
- Trends gradually stabilized after vaccination efforts increased

---

### 4️ Active Case & Fatality Distribution
Donut charts visualize:
- Active cases by month
- Monthly Case Fatality Rate contribution

#### Key Insights
- May recorded the highest active case percentage
- July and August showed elevated fatality rates
- Certain months placed significant pressure on healthcare systems

---

### 5️ Geographic Spread Analysis
An interactive world map visualizes:
- Global case distribution
- Recovery concentration by region

#### Key Insights
- North America, South America, and Asia were heavily impacted
- Larger map bubbles indicate higher pandemic intensity

---

### 6️ Advanced Comparative Analysis
Additional visualizations include:
- Death-to-recovery ratio by country
- Recovery vs case comparison using treemaps
- Countries with unusually high fatality rates

#### Key Insights
- Some countries experienced disproportionately high fatality rates
- Recovery trends varied significantly between regions
- Treemap analysis highlighted dominant contributors to global cases
